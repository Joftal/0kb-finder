# 0KB文件扫描器

这是一个交互式程序，用于扫描指定目录及其子目录中的0KB大小文件，并提供删除功能。

## 使用方法

1. 运行程序：`dotnet run`
2. 在界面中，使用“添加文件夹”按钮选择要扫描的文件夹，或直接拖拽文件夹到窗口中添加到列表。
3. 可以添加多个文件夹。
4. 点击“扫描”按钮开始扫描所有选定的文件夹及其子文件夹。
5. 如果找到0KB文件，将在界面中列出每个文件的路径，每个文件有“打开文件夹”和“删除”按钮。
6. 点击“打开文件夹”按钮打开文件所在目录。
7. 点击“删除”按钮删除文件到回收站。
8. 扫描完成后，点击“重新扫描”按钮回到选择界面，重新选择文件夹。
